Headset-Receiver
================

Custom embedded Arduino board that provides Bluetooth/FM/Jack for a Headphones.
* Bluetooth rn-52 module
* Si4703 FM chip
* Arduino Pro mini 3.3v based board

Repository Contents
-------------------
* **/Eagle Files** - Eagle files for the two boards (.brd, .sch)
